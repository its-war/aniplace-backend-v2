1º Salve toda a estrutura do banco de dados.
   (arquivo incluso: banco_de_dados.sql)

2º Abra o arquivo ".env" e modifique os valores das seguintes variáveis para os valores certos.
	MYSQLHOST | MYSQLUSER | MYSQLPASSWORD | MYSQLPORT | SERVERPORT
   (existem dois arquivos ".env", o ".env.example" não pode ser modificado, jamais)
   (não altere as demais variáveis)

3º Abra a pasta do projeto no terminal e digite o seguinte comando:
	npm install

4º Execute o comando que inicia o servidor:
	npm start
